МЭЖ - dnevnik
*****************************

Содержит функции для отображения страницы регистрации, уроков и главной страницы.

Используемые URL:

- **/auth** - Страница аутентификации
- **/** - Index-страница, главная страница
- **/telegram** - *Только для разработчиков.* Доступ к TG-составляющей(Telegram)
- **/watch_conference** - Просмотр видеоконференций
- **/download_conf** - *Только для разработчиков.* Скачивание окончившейся конференции для сохранения в Базу Знаний.
- **/get_records** - *Только для разработчиков.* Возвращает список находящихся в Базе записей.
