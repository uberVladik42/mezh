База Знаний - uchebnik

Упражнения, записи, сохранения, статистика.

Используемые URL:

- **/create_exercise** - Создание нового Упражнения(Exercise)
- **/redact_exercise/<int:exercise_id>** - Редактирование существующего Упржанения(Exercise)
- **/view_exercise/<int:exercise_id>** - Отображение существующего Упржанения(Exercise)
- **/view_pupil_results/<int:results_id>** - Отображение результатов ученика.
